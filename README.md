Сложно сосредоточиться
Проект "Сложно сосредоточиться" — это адаптивный веб-сайт, посвященный проблемам концентрации внимания в современном мире. Сайт предлагает полезные советы и информацию о том, как лучше фокусироваться на задачах.
Обзор проекта
Сайт представляет собой информационный ресурс с советами по улучшению концентрации. Материал структурирован по разделам, объясняющим причины трудностей с фокусировкой и предлагающим практические рекомендации.

Функциональность
Адаптивный дизайн — корректное отображение на мобильных устройствах, планшетах и десктопах

Переключение тем — поддержка светлой, темной и автоматической темы (в зависимости от системных настроек)

Сохранение выбранной темы — выбор пользователя сохраняется в localStorage

Акцент на читаемости — продуманная типографика и контрастность

Темы оформления
Сайт поддерживает три режима отображения:

Светлая тема — для дневного времени

Темная тема (неон) — с неоновыми акцентами

Авто — автоматически подстраивается под системные настройки

Выбранная тема сохраняется при перезагрузке страницы.
